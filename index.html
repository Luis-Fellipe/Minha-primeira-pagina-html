Surgimento do HTML

(voltar)

HTML foi proposto pela primeira vez por Tim Berners-Lee e Robert Cailliau em 1990, como parte do desenvolvimento do World Wide Web (WWW) no CERN (Organização Europeia para a Pesquisa Nuclear) na Suíça. O HTML original, conhecido como HTML 1.0, foi uma linguagem simples com um conjunto limitado de recursos para marcar a estrutura básica de um documento. Desde então, o HTML evoluiu significativamente. Novas versões foram lançadas, cada uma introduzindo novos recursos, elementos e atributos. A especificação HTML5 foi finalizada em 2014 e trouxe uma série de melhorias e recursos avançados, como suporte para multimídia, elementos semânticos, APIs de JavaScript e muito mais. Desde então, o HTML5 se tornou a versão mais amplamente adotada e usada do HTML. É importante observar que o HTML é uma linguagem em constante evolução, e novas versões e recursos continuam sendo desenvolvidos para melhorar a experiência de desenvolvimento e a funcionalidade das páginas da web.

O que é o HTML?

(voltar)

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto") é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. Apesar de possuir "linguagem" em seu nome, vale ressaltar que HTML não é considerada uma linguagem de programação.

Algumas tags em HTML

(Voltar)

Em HTML, as tags são elementos usados para estruturar e formatar o conteúdo de uma página da web. Elas são escritas entre colchetes angulares (<>) e geralmente vêm em pares de abertura e fechamento. A tag de abertura indica o início de um elemento, enquanto a tag de fechamento indica o fim desse elemento.

Existem muitas tags HTML diferentes, cada uma com sua própria finalidade e significado. Algumas das tags vistas até o momento na DIO.me são:

  • html: Define o início e o fim do documento HTML.
  • head: Contém informações sobre o documento, como o título da página e referências a arquivos externos.
  • body: Contém o conteúdo principal da página que será exibido no navegador.
  • h1, "h2", "h3"..."h6": Cabeçalhos de diferentes níveis.
  • p: Define um parágrafo de texto.
  • a: Cria um link para outra página ou recurso.
  • img: Insere uma imagem na página.
  • ul e li: Cria uma lista não ordenada de itens.
  • ol e li: Cria uma lista ordenada de itens.
  • u: Sublinha uma parte do texto.
  • i: Transforma a caligrafia em molde italico
  • hr:Gera uma linha divisória, como a usada no inicio desta pagina
  • strong:Deixa em negrito
  • mark:Gera uma marcação no texto.

Atributos das tags

(voltar)

Em HTML, os atributos são usados para fornecer informações adicionais sobre as tags e modificar o comportamento ou a aparência dos elementos na página da web. Eles são especificados dentro das tags e geralmente consistem em um nome e um valor, separados por um sinal de igual (=). Alguns exemplos são:

  • id:Especifica um identificador exclusivo para um elemento.
  • src:Usado em tags como "img" e "script", indica a URL ou o caminho do arquivo que deve ser carregado.
  • href:Utilizado em tags como "a" (âncora) e "link", especifica o URL ou caminho do arquivo para o qual o usuário será direcionado quando o link for clicado.
  • width e height::Utilizados em tags como "img" e "canvas", definem a largura e a altura de um elemento, respectivamente.
